# Retail Sales Dashboard using Pandas and Matplotlib

## Project Overview
This project analyzes retail sales data using Python, Pandas, and Matplotlib. The goal is to clean the dataset, analyze sales performance, and create a dashboard that helps business users understand sales trends, regional performance, and product performance.

## Dataset
Dataset used: Sample Superstore / Superstore Sales Dataset  
Source: Kaggle or Tableau Sample Data

## Business Questions
1. Which region has the highest sales?
2. Which category performs best?
3. What is the monthly sales trend?
4. Which products generate the most sales?
5. How does discount affect profit?

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
